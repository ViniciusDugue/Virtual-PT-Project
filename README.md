## About

A research-oriented Unity project that simulates a physical-therapy device patients can interact with in VR. The simulation drives haptic feedback so the patient feels controlled resistance during each exercise, replicating the device's behavior in software.

The goal is to let clinicians prototype and tune therapy programs in VR — adjusting resistance curves, motion ranges, and feedback patterns — before committing to hardware iterations.

## What it explores

- **VR-driven motion capture** — tracking the patient's hand/limb path through an exercise.
- **Haptic resistance** — translating prescribed therapy curves into per-frame haptic force.
- **Exercise scripting** — configurable routines so a clinician can author new programs without touching code.

## Stack

`Unity` · `C#` · `VR` · `Haptics`
